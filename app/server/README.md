# Delicakes Node.js App

## Installation

```bash
npm install
```

## Running the App

### Development mode (with auto-restart):
```bash
npm run dev
```

### Production mode:
```bash
npm start
```

The server will start on http://localhost:3000

## Features

- Serves the Delicakes website
- Static file serving for HTML, CSS, JS
- Image assets served from `/assets` directory
- Health check endpoint at `/api/health`
