
# Start Expense Tracker
This is a simple expense tracker application that allows you to keep track of your expenses.

# Development Setup
To run the application locally, follow these steps:

(From the same location as this README)
1. (**on "app server" terminal**)run `npm install` to install all dependencies.
2. run `npm run build` to start the development server.
3. run `npm run app-server` to start the app server.
4. (**On another terminal"data-api server"**) run `npm run data-api` to start the data API server.
5. Open your browser and navigate to `http://localhost:3000` to access the application.

6. (Optional - **On another terminal "dev watcher"**) To have a continuous development setup, you can execute `npm run watch` to watch for changes and rebuild the project automatically
